# Enviroment Variables - Ortam Değişkenleri

- Uygulamanıun çalıştığı ortama göre değişebilen gizli ve dinamik ayarlardır.
- Gizli bilgileri saklamak için kullanılır
- .env dosyası içerisinde tanımlanır
- `VITE_DEĞİŞKEN_İSMİ=DEĞİŞKEN_DEĞERİ` formatında tanımlanır
- `import.meta.env.VITE_DEĞİŞKEN_İSMİ` yöntemiyle erişilir
